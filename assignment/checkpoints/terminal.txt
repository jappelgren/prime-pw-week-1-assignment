# 8 - Why would you run the command 'git status'?
Running git status displays any files that have been updated past the versions contained in the repository. It will show whether files are need to be staged or are ready to be committed.
# 9 - What does the command 'git add .' do?
git add . will add all files in the current repository that have been modified past their previous versions to be committed. This is called staging a commit.
# 10 - What is the command to push your committed changes to your repo on GitHub?
